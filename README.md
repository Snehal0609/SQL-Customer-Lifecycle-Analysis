# 📊 Customer Lifecycle Analysis (SQL Project)

## 📌 Overview

This project uses SQL to analyze customer data across different lifecycle stages such as acquisition, conversion, retention, and churn.

The goal is to understand customer behavior and generate business insights from structured data.

## ⚙️ Tech Stack

* SQL
* CSV datasets
* Relational tables

## 🗂️ Dataset

* Customers
* Orders
* Order Items
* Products
* Categories
* Channels
